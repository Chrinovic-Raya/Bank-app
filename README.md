#  Simple Banking Application

A console-based banking application written in Java that provides essential banking operations with user authentication and transaction history tracking.

##  Features

- **User Authentication**
  - Secure sign-up and sign-in functionality
  - Password masking during input (when console supports it)
  - Credential validation

- **Banking Operations**
  - Deposit funds
  - Withdraw funds (with overdraft protection)
  - Check account balance
  - View transaction history with timestamps

- **User Interface**
  - Main menu with clear options
  - Mini menu for quick actions after transactions
  - Formatted output for better readability
